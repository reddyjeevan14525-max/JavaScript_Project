# Simple HTML + CSS Page

A clean, responsive static page built using only HTML and CSS.

## Structure
- `index.html` — semantic layout with header, hero, features, gallery, FAQ, contact form, and footer.
- `styles.css` — mobile-first styles with CSS variables, flex/grid utilities, and accessible focus states.

## Run Locally
No build step needed. Just open `index.html` in your browser:

```powershell
Start-Process "d:\JavaScript\Javascript Project\index.html"
```

## Local Development Server

This project includes a simple Express server to serve the static HTML files.

### Prerequisites
- Node.js 16+ installed

### Start the server
```powershell
cd "D:\JavaScript\Javascript Project"
npm install
npm start
```

Then open `http://localhost:3000` in your browser.

### Scripts
- `npm start`: Runs the server
- `npm run dev`: Runs the server with `nodemon` (auto-reload) if installed
